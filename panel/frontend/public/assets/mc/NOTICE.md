# Minecraft UI assets

Текстуры в этом каталоге извлечены из официального Minecraft: Java Edition
client JAR (версия указана в `VERSION`) скриптом `scripts/fetch-mc-ui-assets.sh`.

Minecraft content and materials are trademarks and copyrights of Mojang AB.
This project is not affiliated with, associated with, or endorsed by Mojang AB
or Microsoft. Assets are used exclusively to render this fan-made admin panel
and are not sold or redistributed as a standalone product.

Воспроизведение набора:

```bash
MC_VERSION=release ./scripts/fetch-mc-ui-assets.sh
```

Контрольные суммы извлечённых файлов — в `SHA256SUMS`.
